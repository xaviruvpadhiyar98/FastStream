# FastStream

Watch Movies-TV Show without Downloading with VLC and Python

## Getting Started

You will need -
```
VLC
python3
pip install requests
pip install lxml
sudo apt-get install vlc-plugin-bittorrent
```

Thats It

Run the main.py file
